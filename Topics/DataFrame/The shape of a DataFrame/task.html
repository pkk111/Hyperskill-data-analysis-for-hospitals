Load the .txt file as a Pandas DataFrame, then check how many rows and columns the DataFrame has. Insert the output in the answer field as two numbers separated by a single space. For example, for a DataFrame with 6 rows and 9 columns the answer would be: 6 9.

It can take too long to count all the rows and columns by hand. Recall the name of the attribute that every DataFrame object has for this purpose.


Sample Input:
name,age,education,sex,income
254343,36,1,1,>50k
476429,33,7,1,<50k
363510,33,1,0,>50k

Sample Output:
3 5
